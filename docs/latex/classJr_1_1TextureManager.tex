\doxysubsection{Detailed Description}
Gestionnaire de textures SFML chargé de gérer le chargement, la mise en cache et la libération des textures. 

Cette classe permet de charger des textures depuis le disque une seule fois, et de les réutiliser via un cache interne afin d\textquotesingle{}éviter des chargements redondants coûteux.
