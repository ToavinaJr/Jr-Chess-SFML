Detailed Description

État du jeu affichant l'écran d'aide ou de tutoriel.

Cette classe dérive de GameState et gère l'affichage d'un écran contenant un titre, un contenu textuel explicatif, ainsi qu'un bouton pour revenir à l'état précédent.
