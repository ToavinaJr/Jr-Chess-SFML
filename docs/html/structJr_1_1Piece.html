Detailed Description

Structure représentant une pièce d'échecs logique (sans aspects graphiques).

Contient le type et la couleur de la pièce, avec quelques méthodes utilitaires.

◆ getTextureFileName() [1/2]

std::string Jr::Piece::getTextureFileName ( ) const
inline

Retourne le nom du fichier de texture associé à la pièce.

Le nom est composé d'un préfixe 'w' ou 'b' selon la couleur, suivi d'une lettre représentant le type (P, N, B, R, Q, K), et de l'extension ".png".

Par exemple : "wP.png" pour un pion blanc, "bK.png" pour un roi noir.

Returns
Nom du fichier texture ou chaîne vide si pièce vide.
